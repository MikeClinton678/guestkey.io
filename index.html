The Flow

From booking to door code in four steps.

01

Guest books your property

Hospitable webhook fires. GuestKey creates a registration shell and sends a branded registration link to the guest.

02

Guests register via the link

Each guest verifies their identity — name, address, email code, phone code. Takes 90 seconds. The booker shares the link with their party.

03

All guests registered → code unlocks

The moment the last guest registers, GuestKey pushes a time-bound access code to your smart locks. Instant. No manual work.

04

Guests check in. You relax.

Codes activate at check-in, expire at check-out. You see everything in your admin dashboard. If anything goes wrong, you're alerted.
