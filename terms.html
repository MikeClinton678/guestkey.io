Service Description

GuestKey is a guest experience platform designed for vacation rental hosts. The Service provides:

  • Guest registration and identity verification — Collecting verified contact information from all adult guests via one-time passcode (OTP) verification, not just the primary booker.
  • Security deposit collection and management — Facilitating the collection and release of security deposits through the Host's own connected payment account.
  • Smart home device control — Integration with door locks, thermostats, and other connected devices to provide time-limited access during authorized stay periods.
  • Digital guidebook — A hosted, mobile-friendly property guide including house rules, check-in instructions, local recommendations, and other host-curated content.
  • Noise monitoring integration — Connecting with third-party noise monitoring platforms and surfacing noise level data in the guest portal.
  • Automated multi-channel guest messaging — Scheduled and triggered communications to guests across SMS and other configured channels.
  • Admin dashboard — A web-based interface for Hosts to manage properties, guests, registrations, deposits, and device integrations.
  • Compliance tools — Features to assist Hosts in meeting local guest registration and lodging ordinance requirements.

The Service is accessed through a web browser and requires no app download for guests. Features available to any given Host may vary based on the Host's plan and connected third-party integrations.
